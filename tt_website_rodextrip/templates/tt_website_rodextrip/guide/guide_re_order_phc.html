Quick Repeat Order PHC melalui halaman utama {{name}}
Quick Repeat Order PHC melalui E-mail

Quick Repeat Order PHC melalui halaman Home

Note: Jika order melalui agent maka repeat order dilakukan melalui agent, tetapi jika order dilakukan oleh customer sendiri di {{name}} maka customer langsung dapat melakukan repeat order.

1. Buka halaman utama {{name}}

2. Pilih Product "Test Covid PHC"

3. Tekan tombol Repeat Order

4. Pilih Product "PHC" dan Masukkan Order Number

Order Number didapatkan oleh customer setelah pernah melakukan booking di sistem {{name}} (Order Number dapat dilihat di email atau invoice yang sudah diberikan).

5. Tekan tombol Re Order

6. Pilih Test Type dan Customer yang akan di Re Order

7. Pastikan Data Customer dan Data Schedule Test sudah Benar, lalu lakukan order seperti biasa [SELESAI].

Quick Repeat Order PHC melalui Email

1. Buka kotak masuk email, lalu pilih pesan email Booking PHC atau Issued PHC (judul biasanya terdapat isi seperti PH.***********)

2. Pada isi pesan klik link di Booking Url

3. Tekan tombol Re Order

4. Pilih Test Type dan Customer yang akan di Re Order

5. Pastikan Data Customer dan Data Schedule Test sudah Benar, lalu lakukan order seperti biasa [SELESAI].
